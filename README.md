# AI-Based Loan Approval and Risk Scoring System

A full-stack MERN application integrated with a Python-based AI/ML engine to predict loan approval probability and customer risk score using financial and credit-related parameters.

## Tech Stack
- Client: React.js
- Server: Node.js, Express.js
- Database: MongoDB
- ML Engine: Python, Flask, scikit-learn

## Architecture
Client → Server → ML Engine → Risk Prediction
